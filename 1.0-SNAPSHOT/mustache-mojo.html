mustachifier:mustache

Full name:

com.xebialabs.utils:mustachifier-maven-plugin:1.0-SNAPSHOT:mustache

Description:

Mustachify a Java property file.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: compile.

Optional Parameters

Name Type Since Description
mustache.delimiters String - The delimiter format
Default value is: {{ }}.
fileset FileSet - FileSet including special entries
mustache.generateplaceholder boolean - Configure the mojo to generate a default placeholder value from the key name (mode=placeholders only)
Default value is: true.
mustache.mode Mode - The separator format from the placeholder name and the default value
Default value is: placeholders.
mustache.separator String - The separator format from the placeholder name and the default value
Default value is: :.

Parameter Details

mustache.delimiters:

The delimiter format
  • Type: java.lang.String
  • Required: No
  • Default: {{ }}

fileset:

FileSet including special entries
  • Type: org.apache.maven.shared.model.fileset.FileSet
  • Required: No

mustache.generateplaceholder:

Configure the mojo to generate a default placeholder value from the key name (mode=placeholders only)
  • Type: boolean
  • Required: No
  • Default: true

mustache.mode:

The separator format from the placeholder name and the default value
  • Type: com.xebialabs.maven.mustache.Mode
  • Required: No
  • Default: placeholders

mustache.separator:

The separator format from the placeholder name and the default value
  • Type: java.lang.String
  • Required: No
  • Default: :
